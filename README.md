# Retail Behavior Engine

A professional SaaS dashboard for E-commerce customer segmentation, featuring real-time data processing, K-Means clustering, and interactive data visualization.

##  Key Features
- Machine Learning Engine:Automated K-Means clustering to identify high-value customer segments.
- Interactive UI: Bento-grid layout with dark/light mode and guided onboarding tours.
- Data Export: Seamless CSV export for marketing campaign management.
- Backend: FastAPI-powered API for efficient data normalization and processing.

##  Tech Stack
- Frontend: React.js, Recharts, React-Joyride.
- Backend: FastAPI, Pandas, SQLite.
- Styling: Custom CSS with modern UI/UX design patterns.

##  How to run
1. Backend: `cd backend` -> `uvicorn main:app --reload`
2. Frontend: `cd frontend/dashboard` -> `npm start`
